# SC549_Assignment03_PlayerTracking
SC549 Neural Network Programing Assginment 03 ; Player Tracking in Sports videos using YOLO and Keypoint Detection.
-------------------------------------

## Objective

The objective of this assignment is to perform:

- Player Detection using YOLOv8
- Player Tracking across video frames
- Keypoint (Pose) Detection
- Frame Screenshot Extraction
on multiple short sports video clips
